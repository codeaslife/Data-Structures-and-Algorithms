## 双链表



链表解题的方法论
判断回文单链表
解法一：直接放到栈中，弹出与原链表对比。
解法二：只把右侧的节点放进栈内，弹栈进行对比。
如何实现右半部分放进栈中？
快慢指针：快指针一次2步，慢指针一次1步。

定制指针：
1 2 3 2 1奇数，快指针走完的时候，慢指针在中间
1 2 3 3 2 1偶数，希望出现快指针走完的时候，慢指针在中间的前一位

1->2->3->2->1->null
